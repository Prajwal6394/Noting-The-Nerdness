# Noting The Nerdness

A simple responsive web app for storing your notes and thoughts. It uses your browser's local storage to persist data between sessions.

## Tech Stack
- HTML5
- CSS3
- Bootstrap
- JavaScript
- LocalStorage API
- Visual Studio Code

## How to run:

1. Clone the repository
2. Open `index.html` in a suitable web browser
